Introduction

Adaptiveware is a multi-platform hybrid application development, runtime and build platform. The goal of this project is to devise a common high-level interface for different types of devices - from smart watches, smart glasses and smartphones all the way to smart TVs - and to bind those interfaces to an HTML5-capable browser engine from where they can be accessed using JavaScript. The browser engine is enriched with OS specific implementations of those interfaces that are exposed at DOM level so that web developers can leverage their skills to create hybrid applications using HTML, JavaScript and CSS3 on all of the supported platforms, providing rich user-experiences using a common code-base on a broad spectrum of devices.

Evidently, not all OSes and devices provide have the same software or hardware capabilities but Adaptiveware makes it easy to interrogate the platform at runtime using a common JavaScript API to determine the profile of a device, and the support for certain features, using a unified API across all platforms.

Device types in scope:

  • Smartwatches (Watchfaces/SmallApps created with Javascript, not rendered in HTML5 engine)
    • Pebble
    • Metawatch
    • I'm Watch
    • Sony Smartwatch
  • Smartglasses
    • Google Glass
    • Vuzix Smartwatches (if they're kind enough to donate a license ^^)
  • Smartphones & Tablets
    • Apple iOS (v5 onwards)
    • Google Android (v4 onwards)
    • RIM BlackBerry (v10 onwards)
    • Windows Mobile/Phone (v7.5 onwards)
    • Windows 8/RT (tablet-things)
    • Ubuntu Touch
    • Firefox OS
  • Netbooks
    • Chrome OS
    • Firefox OS
    • Ubuntu Touch
  • Desktop
    • Mac OSX
    • Linux
    • Windows (.NET4 onwards)
  • TVs
    • Samsung SmartTV (2012 onwards)
